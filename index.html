About

브랜드를 직접 만들고, 콘텐츠로 소비자를 설득해 온 마케터입니다.

미국과 호주에서 다양한 문화권의 사람들과 일하며 빠르게 적응하고 협업하는 법을 배웠습니다. 이후 브랜드 ‘파도 프로젝트’를 운영하며 시장조사부터 제품 기획·생산·펀딩·판매까지 브랜드 운영의 전 과정을 직접 경험했습니다.

Problem · Action · Result
Problem
체중 변화가 잦은 여성은 매 시즌 수영복을 새로 사야 했습니다.

기존 수영복은 S/M/L 구조라 임산부·플러스사이즈 여성은 반복 구매가 불가피했습니다.
55~99kg을 아우르는 원사이즈 시장이 비어 있다고 판단했습니다.

Action
제품·콘텐츠·운영을 직접 설계하고 데이터로 개선했습니다.
① 제품 기획중국 공장을 직접 방문해 크링클 소재를 선정하고, 10명 이상 체형의 피팅 테스트를 반복하며 어깨끈·패드·총장 패턴을 보완.
② 마케팅 기획“사이즈 걱정 없이 입는 수영복” USP를 설정하며 와디즈 소통 탭 Q&A를 직접 관리. 브랜드 숏폼 콘텐츠를 100% 직접 기획·제작.
③ 데이터 기반 개선1차 분석 후 컬러 7→4개로 축소해 선택 피로도를 낮추고, 커버업 스타일링 콘텐츠를 추가해 구매 전환을 개선.
④ 팝업 & SNS 운영신촌 팝업스토어를 기획·운영하고, SNS 댓글 이벤트로 인지도 확보 및 수요를 검증.
Result
데이터 기반 개선이 재펀딩에서도 유효함을 증명했습니다.
